## Install
Run the following command in your project directory to install this module.
```bash
tg install testgram/unhandled_reject
```

## Description
A simple listener that emits an event every time there is an
[unhandled promise rejection](https://developer.mozilla.org/en-US/docs/Web/API/Window/unhandledrejection_event) on the page.

These pesky little rejections are often the cause of many async issues at the root cause of infinite loads or hanging pages.
